# 🌍 WorldWise

WorldWise is a modern React application that helps users keep track of the cities they have visited around the world. Users can click anywhere on the interactive map to add a new city, choose the visit date, write personal notes, and build their own travel journal.

---

## 🚀 Features

- 🗺️ Interactive world map powered by Leaflet.
- 📍 Add cities by clicking directly on the map.
- 📅 Select visit dates using a date picker.
- 📝 Save personal notes for every visited city.
- 🌍 Display city location with coordinates.
- 🧭 Client-side routing with React Router.
- ⚡ Fast development experience with Vite.
- 📱 Responsive and clean user interface.
- 🔄 Global state management using Context API and useReducer.
- 🌐 Fetch data from a local REST API using JSON Server.

## ⚙️ Installation

Clone the repository:

```bash
git clone https://github.com/AhmedFathy114/worldwise.git

cd worldwise
```

Install dependencies:

```bash
npm install
```

Start the JSON Server:

```bash
npm run server
```

In another terminal, start the development server:

```bash
npm run dev
```

### Available URLs

- Frontend → http://localhost:5173
- JSON Server API → http://localhost:9000
